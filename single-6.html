Generative AI in Action: Explore My Blog Post Generator

I've built a Blog Post Generator using the GPT-2 large model from Hugging Face, and I've deployed it on Hugging Face Spaces using Gradio. This project showcases the potential of generative AI in creating coherent and creative content based on user input.

Here's a quick overview:

  • Model: GPT-2 Large from Hugging Face
  • Framework: Gradio for the user interface
  • Deployment: Hugging Face Spaces
  • Features: Generates blog posts up to 500 words with controlled creativity and coherence
