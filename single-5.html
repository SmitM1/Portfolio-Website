Innovating with AI: Our Speech Processing Project Unveiled

I'm thrilled to announce the completion of my latest project: a Flask web application that leverages cutting-edge AI models for speech-to-text and text-to-speech functionalities!
We've fine-tuned Wav2Vec2 and SpeechT5 models for seamless speech-to-text and text-to-speech functionalities.

Key Features:

  • User authentication for secure access
  • Upload audio files for transcription
  • Real-time text-to-speech synthesis
  • Storage of transcriptions and synthesized audio

Technologies Used:

  • Flask
  • Hugging Face Wav2Vec2 and SpeechT5 models
  • HTML/CSS/JavaScript for the front-end
  • MongoDB for database and PyMongo for database management
  • Python for backend scripting
